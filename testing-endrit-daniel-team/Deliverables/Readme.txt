Used JUnit 4
IDE Used: VSCode
Open JDK version: 11.0.22

When executing test methods, we used to 'test' button on the left side of the program next to each test method header. 

To run the program in terminal with your own test.txt file, we did:
javac Printtokens.java <----- to compile
java Printtokens test.txt <----- to run with your test file
